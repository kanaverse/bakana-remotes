Remote readers for bakana

Overview

This package provides a variety of remote readers for the bakana package, allowing us to perform scRNA-seq analyses on datasets from other sources. Currently we support:

  • Count matrices and row/column DataFrames from ExperimentHub.

In the future, we aim to support:

  • Count matrices and row/column annotations from ArtifactDB.

Quick start

Usage is as simple as:

import * as remotes from "bakana-remotes";
import * as bakana from "bakana";

// Add desired readers for unserialization.
bakana.availableReaders["EperimentHub"] = remotes.ExperimentHubDataset;

For ExperimentHub-sourced datasets, we can construct the following object:

let ehub = new remotes.ExperimentHubDataset("zeisel-brain");

This can then be used in an entry of datasets in bakana.runAnalysis(). The supported set of IDs is listed in ExperimentHubDataset.availableDatasets()... which is currently not a lot right now.

Links

See the bakana documentation for more details on how to create a custom Dataset reader. Implementations of readers for other databases are welcome.

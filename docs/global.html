Load a sparse external:ScranMatrix given an ArtifactDB ID. This currently supports loading from hdf5_dense_array, hdf5_sparse_matrix and hdf5_delayed_array instances.

Properties
Name Type Attributes Default Description
getFun function <optional>
<nullable>

Function that accepts a URL string and performs a GET to return a Response object, see getFileMetadata for details.
downloadFun function <optional>
<nullable>
 null

Function that accepts a URL string and downloads the resource, returning either a file path (string) or a Uint8Array containing the file contents; see getFile for details.
forceInteger boolean <optional>
 true

Whether to force an integer representation for the matrix data. This saves space when floating-point is unnecessary.
layered boolean <optional>
 true

Whether to use a layered representation for the sparse matrix. This saves space but adds complexity in downstream analysis by permuting the rows.
Returns:

An object containing:

  • matrix, a external:ScranMatrix containing the contents of the matrix.
  • row_ids, an Int32Array containing the row identities, possibly permuted when layered = true.
